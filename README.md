# 🌤️ Weather App

A modern, fast, and responsive Weather Application designed to provide real-time weather updates with a clean and intuitive user interface. Built purely with Vanilla JavaScript, HTML5, and CSS3.

## ✨ Features

- **Real-Time Data:** Fetches accurate weather conditions, temperature, humidity, and wind speed.
- **Premium UI/UX:** Features a vibrant gradient background, smooth glass-like components, and modern typography.
- **Fully Responsive:** Perfectly optimized for all screen sizes (Desktop, Tablet, and Mobile).
- **Dynamic Icons:** Weather icons update automatically based on the current weather conditions.
- **Entrance Animations:** Smooth CSS animations for a professional feel when the app loads.

## 🛠️ Tech Stack

- **HTML5:** Semantic markup and structure.
- **CSS3:** Flexbox, media queries, gradients, and keyframe animations.
- **JavaScript (ES6):** DOM manipulation, event handling, and asynchronous API calls (Fetch API).
